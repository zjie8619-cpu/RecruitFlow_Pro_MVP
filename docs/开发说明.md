# 开发说明

技术栈

Python 3.10+、SQLite、Pandas、Streamlit（展示层）

离线/LLM

默认 offline 规则，脱网可跑。

如接入 Claude/OpenAI：在 backend/configs/model_config.json 将 llm_provider 改为 claude/openai，并设置 ANTHROPIC_API_KEY/OPENAI_API_KEY 环境变量；在 backend/core/ 中扩展调用逻辑（已留接口位置）。

表结构

jd、resume、score、audit（见 backend/storage/db.py）

风控实现

盲筛：展示层隐藏敏感字段。

阈值：confidence_threshold 以下标记 blocked_by_threshold=True，不自动发邀请。

版本：backend/utils/versioning.py 快照配置与模板。

审计：所有操作写入 audit。
