🚀 一键启动程序（3种方法）

方法1：双击启动（最简单）⭐⭐⭐
----------------------------------------
1. 找到项目根目录的 "启动程序.bat" 文件
2. 双击运行
3. 等待浏览器自动打开 http://localhost:8501

方法2：PowerShell 启动
----------------------------------------
1. 右键点击 "启动程序.ps1"
2. 选择 "使用 PowerShell 运行"
3. 等待浏览器自动打开

方法3：命令行启动
----------------------------------------
打开 CMD 或 PowerShell，运行：
  启动程序.bat

或直接运行：
  .venv\Scripts\python.exe -m streamlit run app/streamlit_app.py --server.port 8501

========================================
📋 首次使用前准备
========================================

1. 安装 Python 3.8+
   - 下载：https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. 创建虚拟环境（如果还没有）
   命令行运行：
   python -m venv .venv

3. 安装依赖
   命令行运行：
   .venv\Scripts\pip install -r requirements.txt

4. 配置 AI Key（可选，用于 AI 功能）
   在项目根目录创建 .env 文件，添加：
   SILICONFLOW_API_KEY=your_api_key
   或
   OPENAI_API_KEY=your_api_key

========================================
❓ 常见问题
========================================

Q: 启动后浏览器没有自动打开？
A: 手动访问 http://localhost:8501

Q: 端口被占用？
A: 启动脚本会自动处理：
   - 自动关闭旧进程
   - 如果无法关闭，自动使用端口 8502
   - 手动处理：关闭其他 Streamlit 应用

Q: 导入错误？
A: 确保已安装所有依赖：pip install -r requirements.txt

Q: AI 功能不可用？
A: 检查 .env 文件中的 API Key 配置
